# Update k8s cluster certificate

**1.먼저 각 master node와 worker node의 기존 인증서를 삭제한다.**

```
# ssh to master node
# rm -rf /opt/kubernetes/pki
# rm -rf /etc/kubernetes/pki

```

2**.인증서를 재 생성한다.**

```
# cd cubescripts
# ansible-playbook -i inventories/inventory -u root cert.yaml
```
